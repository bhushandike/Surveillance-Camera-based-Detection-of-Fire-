# Surveillance-Camera-based-Detection-of-Fire-
The proposed system is designed to develop a machine learning-based surveillance camera system for 
real-time fire detection. By analyzing video feeds from cameras, it utilizes computer vision 
techniques to identify flames and smoke. Upon detection, the system triggers an alert mechanism, 
notifying relevant authorities or personnel via email for prompt action. This solution aims to enhance 
safety and minimize damage caused by fire incidents, focusing on accuracy and reliability across 
various environments.
